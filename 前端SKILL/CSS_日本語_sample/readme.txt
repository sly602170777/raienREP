▽grid-guide.cssについて

grid-guide.cssはボックスの構造をわかりやすく表示するために用意したスタイルシートファイルです。

・クラス名が「box」で始まる要素を枠で囲み、ボックスの構造を視覚化します。

・ボックスの階層によって色が変わります。
	最上位階層：赤
	子階層：黄色
	孫階層：青

・ボックス内に「BOX1」といった文字を入れると、ボックスの高さが200pxに固定して表示されます。

・ボックス内に<div>などでマークアップしたパーツを入れると、200pxの高さは解除され、パーツに合わせた高さになります。
